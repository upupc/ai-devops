# Project Artifacts 目录说明

本目录用于存放项目相关文档和规范的生成模板。

## 目录结构

### product/
产品相关文档模板目录，用于存放产品层面的需求、规划和业务基线文档模板。

包含内容：
- 产品白皮书 (Product White Paper)
- 产品需求文档 (PRD)
- 产品规格说明

### tech/
技术相关文档模板目录，用于存放技术实现、架构设计和开发相关的文档模板。

包含内容：
- 技术规格书 (Technical Specification)
- 技术方案 (Technical Plan)
- 架构设计文档

## 使用说明

该目录下的模板用于快速生成标准化的项目文档，确保文档格式统一、内容完整。大模型可以根据这些模板生成相应的项目文档。
